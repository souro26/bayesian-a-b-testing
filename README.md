# argonx

Bayesian decision engine for A/B testing.

## Overview

`argonx` is a Python library designed for robust Bayesian A/B testing, integrating primary metrics with guardrails and sequential stopping rules.

## Installation

```bash
pip install argonx
```

## Features

- **Bayesian Models**: Support for Binary, Count, Gaussian, and LogNormal models.
- **Composite Scoring**: Integration of multiple metrics into a single decision score.
- **Sequential Testing**: Stopping rules based on expected loss and probability of being best.
- **Visualization**: Built-in plots for posterior distributions, lift, and sequential trajectories.
